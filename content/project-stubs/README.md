# Project Stubs

These Markdown files are drafting scaffolds for richer project pages.

They are not rendered by the public site yet. Use them to gather source
material, then convert mature drafts into `content/projects.json` records and
public project pages.

Each stub should collect:

- short public summary;
- people, HQP, and collaborator names;
- funding and partner context;
- publications, reports, software, web apps, and datasets;
- candidate images or figures with provenance;
- public links and private-source notes;
- open questions for the maintainer.
